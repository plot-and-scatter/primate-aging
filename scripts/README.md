# Data Processing Scripts

This directory contains Python scripts for processing and transforming raw data into formats suitable for the web visualization.

## Setup

1. Create a virtual environment:
   ```bash
   python3 -m venv venv
   ```

2. Activate the virtual environment:
   ```bash
   # On macOS/Linux:
   source venv/bin/activate

   # On Windows:
   venv\Scripts\activate
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Data Pipeline

```
data/raw/        → Raw data files (input)
scripts/         → Processing scripts
data/processed/  → Processed data (output)
static/data/     → Final data for web app
```
